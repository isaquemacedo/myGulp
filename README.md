# myGulp

Comandos para instalação, porém se for a primeira vez que está utilizando gulp e bower, os instalem globalmente:

```sh
$ npm install -g gulp
$ npm install -g bower
```

No diretório do projeto, execute os comandos:

```sh
$ npm install
$ bower install
```
